SubclassesDynamicBinningDefinition, DynamicColumn, DynamicFillStyle, DynamicKeyFilter, DynamicKeySet, DynamicLineStyle, DynamicPlotter

This object links to an internal ILinkableObject. The internal object can be either a local one or a global one identified by a global name.

This function will call lockObject() on the ILinkableHashMap that contains the internal object. This object will also be locked so that no new objects can be requested.

removeObject()method 
public function removeObject():void

If the internal object is local, this will remove the object (unless it is locked). If the internal object is global, this will remove the link to it.

requestGlobalObject()method 
public function requestGlobalObject(name:String, objectType:Class, lockObject:Boolean):*

This function creates a global object using the given Class definition if it doesn't already exist. If the object gets disposed of later, this object will still be linked to the global name. If the existing object under the specified name is locked, this function does nothing.

Parameters
name:String — The name of the global object to link to.
 
objectType:Class — The Class used to initialize the object.
 
lockObject:Boolean — If this is true, this object will be locked so the internal object cannot be removed or replaced.

Returns
* — The global object of the specified name and type, or null if the object could not be created.
