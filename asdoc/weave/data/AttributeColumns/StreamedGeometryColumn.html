metadataRequestModeconstant 
public static const metadataRequestMode:LinkableString

This mode determines which metadata tiles will be requested based on what geometry data is requested. Possible request modes are:
all -> All metadata tiles, regardless of requested X-Y-Z range
xy -> Metadata tiles contained in the requested X-Y range, regardless of Z range
xyz -> Metadata tiles contained in the requested X-Y-Z range only
