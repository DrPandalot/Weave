getSingletonInstance()method 
public static function getSingletonInstance(singletonInterface:Class):*

This function returns the singleton instance for a registered interface. This method should not be called at static initialization time, because the implementation may not have been registered yet.

Parameters
singletonInterface:Class — An interface to a singleton class.

Returns
* — The singleton instance that implements the specified interface.
initializeExternalInterface()method 
public static function initializeExternalInterface():void

This function will initialize the external interfaces so calls can be made from JavaScript to Weave. After initializing, this will call an external function weaveReady(weave) if it exists, where the 'weave' parameter is a pointer to the instance of Weave that is ready.
